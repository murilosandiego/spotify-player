# Boilerplate JS with TDD

Boilerplate developed in JS with TDD course

### Installing

```
git clone https://github.com/murilosandiego/js-tdd-boilerplate.git
cd js-tdd-boilerplate
rm -rf .git
git init
npm install
npm run test:coverage
```
